# teamA-api

## 環境構築方法

- 下記コマンドを実行する

```
make init
```

- 以下 URL が api のパス <br>
  localhost:9010

## make コマンド一覧

| コマンド     | 説明                                   |
| ------------ | -------------------------------------- |
| make init    | 環境構築時に最初に行うコンテナ立ち上げ |
| make down    | コンテナを落とします                   |
| make api     | コンテナの中に入ります                 |
| make analyse | 静的解析とフォーマッターを動かします   |
